# Calculator Project
### By Parker Hubley and Carter Jordan

## Premise
- Make a calculator that can handle basic operations.
- After inputting an equation, a result should come out.

## Features
- Tip Calculator - Calculates Tip.
- History - Shows recent calculations.
- Unit Converter - Converts units in length, speed, area, weight, and temperature.
- Operators - +, -, x, /, %.
- More - All clear, delete, switch sign, decimals.

## Pseudocode
### Main Calculator
- Once a number on the calculator is pressed, it is then displayed on the top panel. It is also stored in the expression.


### Converter
- The converter required a lot more code, so we made a separate GUI, engine, and controller.
- There are combo boxes that let you choose which category of converter you want, and then the units you want to convert it into.
- You then enter the amount and press the bottom button.
- It then displays what you entered, what the output was, and the units for both.

### Tip Calculator
- Input the information: the amount of the bill, the tip percentage, and the amount of people splitting it.
- Then press calculate and it displays the tip amount, the total cost, and the amount per person.

### History
- Once equals is hit, the expression is then displayed in the history section.
- There is a clear button that clears the history when pressed.
